# About Me — Cyber theme

Situs statis sederhana "About Me" dengan tema cyber / neon. Termasuk:

- `index.html` — halaman utama
- `css/styles.css` — gaya dan visual
- `js/script.js` — efek ketik, copy email, tahun otomatis
- `assets/bg.svg` — grafis latar

Cara menjalankan: buka `index.html` di browser, atau jalankan server lokal dengan `npm run server`.

Ganti placeholder: ubah "Nama Kamu" dan alamat email di `index.html`.
